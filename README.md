 To-Do List - Documentação do Projeto
📖 Visão Geral
O To-Do List é um sistema de gerenciamento de tarefas que permite aos usuários cadastrar-se, autenticar-se e realizar operações sobre suas tarefas. O projeto foi desenvolvido em Java com Spring Boot e utiliza um banco de dados MySQL, alocado em um container Docker.

🎯 Objetivo
Facilitar a organização de tarefas diárias, permitindo que o usuário:
- Cadastre-se e faça login para gerenciar suas tarefas.
- Adicione, edite, exclua e visualize suas tarefas.
- Marque tarefas como concluídas.
- Armazene dados de maneira segura.

🛠 Tecnologias Utilizadas
- Java (Spring Boot) – Backend
- MySQL – Banco de dados
- Docker – Containerização do banco de dados
- Maven – Gerenciamento de dependências
- Postman – Teste de API
- Git/GitHub – Controle de versão e hospedagem de código

⚙️ Configuração do Ambiente
Antes de executar o projeto, certifique-se de ter instalado:
- Java 21+ e Maven configurados.
- Docker para rodar o banco de dados MySQL.
- Git para versionamento do código.
